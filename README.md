# reactjs-iscroll

基于 iscroll 实现的 react 组件

## 引入

```
npm install reactjs-iscroll --save
```

## 查看例子

从 github 上 clone 代码后

```
npm install
gulp example
```

在浏览器中打开 http://localhost:9090

## 自定义开发

```
gulp build
```
